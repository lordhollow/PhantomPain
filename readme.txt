■主な機能

・レスメニュー
	レスの上にマウスカーソルを置いてちょっと待つと、右上にメニューが付く。
	メニューの項目はレスによって出たり出なかったりする。

・レスアンカーのアンカー先ポップアップ
	レスアンカーにマウスカーソルを置いてちょっと待つ。
	取得済みレスであれば、表示範囲外のレスであってもポップアップする。

・IDポップアップ
	複数発言のあるIDのみ、IDCOLORによる色づけを行なう。
	IDをクリックすることで、当該IDの発言をポップアップする。
	このポップアップはもう一度IDをクリックすると消える。

・URLのリンク先ポップアップ
	通常のURLリンクにマウスカーソルを置いてちょっと待つ
	リンク先のデータによって表示されるポップアップは異なる。
	画像や動画の場合：サムネイルを表示
	既読スレッドのURLの場合：スレッドタイトルを表示
	未読スレッドのURLの場合：板名を表示（するかもしれない）
	その他：SimpleAPIによるページイメージの表示

・このレスにレス
	レスメニューのResToマークをクリックすることでそのレスにレスするための書き込みウィザードを表示する。
	{Alt+Enter}を押すことでレスメニューが表示されているレスに対するレスを行なう。


・逆参照
	あるレスにアンカーが刺さっている場合、そのレスの番号が太字になる。
	この状態では、レスメニューの以下の機能が使用可能。
	ポップアップ：逆参照情報をポップアップする。もう一度押すと消える。
	ツリー状に展開：逆参照情報を元に関連するレスをツリー状に表示する。もう一度押すと消える。
	抽出： 抽出モードに切り替わり、そのレスとそれを参照するレスが表示される

・粘着（トラッキング）
	レスメニューの「Tracking]マークをクリックすることで、
	粘着指定したレスのIDとTripから、同一人物の発言と思われるものを全てマークする。
	もう一度クリックするとこの状態は解除される。
	Trackingは別スレッドにも適用される。また、すでに開かれている別のタブにも影響する。
	3つまでのトラッキングは色によって識別できる。
	設定画面からトラッキングの解除を実施することもできる。

・しおり
	レスメニューのしおりマークをクリックすることでそのレスを「しおり」にする。
	すでにしおりであるレスでは、しおりを解除できる。
	しおりに指定されているレスはスレッドメニューで確認でき、それをクリックすると該当レスにジャンプする。
	表示範囲外の場所にしおりがあるなら、そこまで追加ロードしてジャンプする。

・ピックアップ
	レスメニューのPickupマークをクリックすることでそのレスをピックアップできる。
	すでにピックアップされている場合は、ピックアップ状態を解消する。
	ピックアップしたレスはスレッドメニューからポップアップ表示または抽出できる。

・画像のサムネイル展開
	レスメニューの画像マークをクリックすることで、そのレスにある画像のURLから取得したサムネイルを表示する。
	展開後にもう一度クリックするとサムネイルは消える。


・ギアモード（前後のレスを参照）
	レスメニューのギアアイコン（上下矢印）をクリック（またはアイコン上でホイール操作）することで
	「ギアモード」の青いポップアップが表示される。
	ギアモードのポップアップにはレスが１つのみ表示され、ホイール操作によってその前後のレスに表示を切り替えることができる。

・あぼーん関連
	あぼーんの即時適用は有効。
	あぼーんされたレスは非表示ではなく、見えるんだか見えないんだか微妙な色合いで小さく表示される。

・追加ロード
	レスメニューのMoreで現在表示されているレスの後ろに続くレスを読み込む。
	同MoreBackで現在表示されているレスの前に続くレスを読み込む。
	スクロールバーが上下の端に張り付いているとき、さらにスクロールし続けることでも追加ロードされる。
	追加ロード時は、既読レスが残っているときは設定した数だけ読み出しを行なう。
	既読レスが残っていないときは、新着のチェックを行ない、すべての新着レスを表示する。
	
	自動更新も可能。
	自動更新モードをONにすると、定期的に新着レスをチェックする。
	自動更新モード中はMoreアイコンが回転する。

・テンプレート参照
	スレッドメニューのテンプレートボタンをクリックすることでテンプレートを表示する。
	設定により、１〜ｎ個表示するか、１をギアモードで表示することができる。

・抽出（検索）
	スレッドメニューの検索ボタンにより、抽出モードを切り替えることができる。
	抽出モードでは背景が赤くなる。
	抽出条件のポップアップに入力した内容に合致するレスのみ表示される。
	検索条件は単純一致または正規表現で記述する。
	検索対象をピックアップされたものだけに制限することもできる。

・画像ビュアー
	スレッドメニューの画像ビュアーボタンにより、ビュアーモードになる。
	ビュアーでは画像のスライドショーができる。

・新着マーク
	新着レスは赤っぽい。
	新着があるとき、スレッドメニューに赤！が表示される。
	それをクリックすると新着レスの先頭にジャンプ。

・板名表示
	２ｃｈの場合、板名をスレッド一覧に表示する。
	この内容はboadname.cssで設定されている。
	この機能を使用しない場合はstyle.cssの@import url("boardname.css")の行を削除する。
	★別ファイルで指定してそれをロードするようにしたほうがよさげ

・ポップアップの消し方
	ポップアップからマウスカーソルを出す
	もともと外にあるときは一度中に入れる
	設定項目「連鎖的なポップアップの破棄」をONにすることで、
	ポップアップからポップアップを表示した場合、
	元になっているポップアップが消えた時点で、
	そこからポップアップした全てのポップアップを閉じる。
